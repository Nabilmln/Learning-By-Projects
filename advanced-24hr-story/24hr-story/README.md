24hr Stories

A frontend Instagram Stories clone built with React and Vite.

Features
✔ Upload stories
✔ Automatic image resize
✔ Local Storage
✔ Story expires after 24 hours
✔ Swipe navigation
✔ Keyboard navigation
✔ Dark Mode
✔ Responsive UI
